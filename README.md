## 📘 Descripción general del sistema

Este proyecto corresponde a la actividad sumativa semana 3 de la asignatura *Desarrollo Orientado a Objetos I*. Sistema básico compuesto por múltiples clases relacionadas desarrollado en Java, cuyo objetivo es modelar y gestionar entidades representativas de una agencia de turismo llamada Llanquihue tour. La aplicación se desarrolla aplicando los principios de encapsulamiento, composición, herencia, polimorfismo e interfaces.

El proyecto fue desarrollado a partir del caso de la agencia de turismo, cuyo requerimiento es un sistema que permita representar estructuradamente las entidades principales de su ecosistema, automatizar tareas repetitivas y facilitar futuras expansiones funcionales. Proponemos una solución estructurada, modular y reutilizable.

## 🧱 Estructura general del proyecto

```plaintext
📁 src/
├── App/       		    # Clase principal LlanquihueTour contiene el método main.
├── Model/     		    # Clases de dominio (Persona, Empleado, Dirección, Rut)
├── ExceptionPersona	# Clase para manejo de excepciones de la clase Persona.

```
